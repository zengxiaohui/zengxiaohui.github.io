阿里大于jar包导入maven 一开始想倒入本地jar包，直接使用了导入本地jar包，发现并不可以，因为是maven统一管理jar包，所以直接导入是不可以的。然后在网上看了很多办法，包括在终端安装jar包，但是发现自己的命令没有，要安装maven才行（我是intellij自带maven），捣鼓了好久，其实我只是想加入一个本地jar包，做了很多麻烦的事情发现都没用。 最后我在WEB-INT里面加入了lib文件夹，把我的jar包放在文件夹里。并且在pom.xml里面加入了以下语句： my-jar my-jar 1.0 system ${project.basedir}/src/main/webapp/WEB-INF/lib/taobao-sdk-java-auto_1455552377940-20160607.jar org.apache.maven.plugins maven-shade-plugin make-assembly package shade jar-with-dependencies xxx-jar-with-dependencies
